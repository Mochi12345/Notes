How to create a Git note
	1) open folders
	2) go to documents
	3) open notes
	4) create a text document
	5) type your notes
	6) save
 	7) right click the notes
	8) Tortoise Git
	9) click add
	10) commit
	11) type comment
	12) push
	13) close
Using Git From Command Prompt
	1)cd C:\Users\Jennifer\Documents\Notes               **Goes to directory
	2)git diff               **Difference w/ current and the one on github
	3)git add "[File Name]"               **Put the modified onto github
	4)git status               **deleted, modified, changed
	5)git commit -m "[message]"               **Confirms changes and adds the message
	6)git push               **Finalizes the notes and adds it to repository
	7)git pull               **gets existing notes and brings its up
	8) git clone [URL]        ** copies new code and keeps it
	
How to grab a new Repo using Command Prompt
	1)open command line
	2)create new directory using mkdir 
		'mkdir test2'
	3) git clone [URL] to add repo to directory
		'git clone https://github.com/Mochi12345/Notes.git'
	4) can open in text editor by typing in the file
		'~/test2/Notes/Git.txt'		(~ = home directory)
